# Library-Management-System

 To run: 1)Install node on your PC.<br>
 2)Make a folder (eg:lib management sys), then save all these files<br>
 3)cd to the folder<br>
 4)>node server.js<br>
 5)Go to localhost:5000/ you can use the website!<br>
